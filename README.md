# Web Vitals Lab

A comprehensive hands-on lab for understanding, simulating, and fixing every Core Web Vital.

## Live Site
https://rsujathanr.github.io/web-vitals-lab/

## Scenarios
- 🖼️ Images — LCP, CLS
- ⚙️ JavaScript — INP, TBT, FCP
- 🎨 CSS — FCP, LCP, TBT
- 🔤 Fonts — CLS, FCP
- 📐 Layout Stability — CLS
- 🌐 Network & Resource Hints — TTFB, LCP
- 💾 Caching — TTFB, LCP
- 🔌 Third-Party Scripts — TBT, FCP, INP

## How to Use
Each scenario has a `bad.html` (anti-patterns) and `good.html` (optimized).
Open both, compare the live metrics overlay in the bottom-right corner.

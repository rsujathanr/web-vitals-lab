Yielding to the Browser

No scripts in <head>. Immediate visual feedback on click before any heavy work starts. Long tasks split into 50ms chunks using a scheduler.yield() polyfill. Search debounced to a single task per pause.

Live Demo

The page parsed in calculating…. Click the button — it gives immediate feedback then does the same 2s of work in chunks. Try typing in the search box while chunks run.

✅ What's fixed

  • Zero scripts in <head> — HTML parses and FCP happens in milliseconds, not 1.2s.
  • Button click updates state instantly (visual feedback before work starts) — the browser can paint the response frame, keeping INP low.
  • The 2s of work is split into 50ms chunks. Between each chunk, setTimeout(0) yields to the browser — input events can be processed.
  • Search input is debounced to 250ms — only one task fires per pause, not one per keystroke.

📐 Before → After

  • Scripts in <head> blocking 1.2s → zero scripts in head
  • 2000ms sync task on click → immediate feedback + chunked work
  • 300ms block per keypress → 250ms debounce, one task total
  • INP > 2000ms → INP < 50ms
  • TBT 1200ms → TBT ~0ms (no long tasks in head)

// ✓ scheduler.yield() polyfill function yieldToMain() { if ('scheduler' in window && 'yield' in scheduler) { return scheduler.yield(); } return new Promise(function(r) { setTimeout(r, 0); }); } // ✓ Immediate feedback + chunked work btn.addEventListener('click', async function () { btn.textContent = 'Working…'; // instant visual feedback → low INP btn.disabled = true; await yieldToMain(); // yield so browser can paint the feedback var CHUNKS = 40; // 40 chunks × 50ms ≈ 2s of work for (var i = 0; i < CHUNKS; i++) { var deadline = Date.now() + 50; while (Date.now() < deadline) { /* work */ } updateProgress(i / CHUNKS); await yieldToMain(); // yield between each chunk } btn.textContent = 'Done ✓'; btn.disabled = false; }); // ✓ Debounced search — one task per pause var debounceTimer; search.addEventListener('input', function () { clearTimeout(debounceTimer); debounceTimer = setTimeout(function () { runSearch(search.value); // fires once, 250ms after typing stops }, 250); });

Critical CSS Inline & Composited Animations

All critical CSS inlined in <style> — zero blocking requests. Non-critical CSS deferred via preload trick. Ball animates with transform + will-change. Cards use explicit transition properties. Off-screen section uses content-visibility: auto.

Off-screen section (content-visibility: auto)

This section is painted only when it enters the viewport. content-visibility: auto skips layout, paint, and compositing for off-screen content — reducing Time to Interactive and First Paint for long pages by up to 7× (Google found this on their Japan site).

✅ What's fixed

  • Zero external stylesheets in <head> — all critical styles are inline, so FCP happens in milliseconds with zero network wait.
  • will-change: transform on the ball — browser promotes it to its own GPU layer before the animation starts.
  • transform: translateX() not left — runs on the compositor thread, never triggers layout.
  • Explicit transitions only: border-color, box-shadow, opacity — all compositor-safe.
  • content-visibility: auto on the off-screen section — skipped entirely until it scrolls into view.

📐 Before → After

  • 2 blocking stylesheets (286KB) → 0 blocking requests
  • left animation → transform: translateX()
  • transition: all → transition: border-color .2s, box-shadow .2s
  • filter: drop-shadow → box-shadow (compositable)
  • No paint skipping → content-visibility: auto

<!-- ✓ Critical CSS inlined — zero blocking requests --> <style> /* only above-the-fold rules */ body { background: #0d0f18; color: #e4e6f0; ... } </style> <!-- ✓ Non-critical CSS: preload trick → non-blocking --> <link rel="preload" href="styles.css" as="style" onload="this.rel='stylesheet'"> /* ✓ GPU-composited animation — no layout */ .ball { will-change: transform; animation: slide 2s linear infinite; } @keyframes slide { to { transform: translateX(calc(100% - 40px)); } } /* ✓ Explicit transitions — no layout properties */ .card { transition: border-color .2s, box-shadow .2s; } /* ✓ Skip off-screen paint entirely */ .below-fold-section { content-visibility: auto; contain-intrinsic-size: auto 300px; }

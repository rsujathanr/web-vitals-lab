❌ What's wrong

  • Images have no width/height — browser reserves 0px, then expands when the image loads, pushing all content below.
  • Ad slot starts at height: 0 — when JS expands it to 90px, everything below shifts down (large CLS event).
  • Cookie banner uses position: static flow — when it appears, it pushes page content upward.

✅ How to fix → good.html

  • Add width/height + aspect-ratio wrapper on every image — exact space reserved before bytes arrive.
  • Give the ad slot min-height: 90px from the start — filling it never changes the layout.
  • Use position: fixed for the cookie banner — overlays the page without shifting flow content.
